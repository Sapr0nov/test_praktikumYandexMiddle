## Commands 💬

#### clear cache and user's data

```sh
npm run clean
```

#### build project

```sh
npm run build
```

#### launch server

```sh
npm start
```

#### linter

```sh
npm run lint
```

#### prettier format

```sh
npm run lint:format
```

#### TESTS

```sh
npm run test
```
